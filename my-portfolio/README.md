# Lab 5 – React Portfolio

This lab is an individual project that implements a personal portfolio website using React, React Router, and Bootstrap. The site includes multiple pages, reusable components, accessibility considerations, and a hosted deployment link.

## Built With

* Vite – Used as the build tool and development server for the React application.
* React – Used to build the component-based front-end.
* React Router – Used for client-side routing between the Home, About, Projects, and 404 pages.
* Bootstrap 5 – Used for layout, spacing, and UI styling such as the navbar and cards.
* Testing Library + Vitest – Used to write basic unit tests to verify component rendering.
